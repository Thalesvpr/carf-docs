## Guias Rápidos

### Instalar a biblioteca

```bash
bun add @carf/ui @carf/tscore
```

### Rodar Storybook

```bash
cd PROJECTS/LIB/TS/UI-COMPONENTS/SRC-CODE
bun install
bun run storybook
```

### Publicar nova versão

```bash
bun run build
bun test
npm version patch
git push --tags
npm publish
```
