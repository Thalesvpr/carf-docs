# PROJECTS

Implementações técnicas do sistema CARF. Cada projeto tem código fonte em SRC-CODE e documentação em DOCS.

O [backend .NET](./GEOAPI/README.md) fornece a API REST com dados geoespaciais. O [portal web](./GEOWEB/README.md) é a interface para analistas e técnicos. O [app mobile](./REURBCAD/README.md) permite coleta de dados em campo mesmo sem internet. O [plugin QGIS](./GEOGIS/README.md) é usado para análises espaciais avançadas. O [console admin](./ADMIN/README.md) gerencia usuários e permissões. A [autenticação](./KEYCLOAK/README.md) é feita via Keycloak com OAuth2/OIDC. O [portal de documentação](./WEBDOCS/README.md) contém guias para usuários finais. E as [bibliotecas compartilhadas](./LIB/README.md) fornecem código reutilizado entre os projetos.
