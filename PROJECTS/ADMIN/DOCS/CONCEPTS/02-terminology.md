# Terminology - ADMIN

## Glossário

**SPA (Single Page Application)** - aplicação web que carrega uma única página HTML e atualiza dinamicamente via JavaScript. **TanStack Query** - biblioteca para server state management com cache, refetch, mutations. **Server State** - dados que vivem no servidor (users, tenants) gerenciados por TanStack Query. **Client State** - dados locais do navegador (UI flags, modals) gerenciados por Zustand. **Query** - busca dados do servidor com cache automático. **Mutation** - modifica dados no servidor invalidando cache. **Protected Route** - rota que valida auth/role antes de renderizar. **RBAC** - Role-Based Access Control. **Proxy Seguro** - intermediário que isola secrets do frontend.
