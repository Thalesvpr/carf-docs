# VALUE-OBJECTS

Objetos de valor imutáveis sem identidade própria comparados por valor ao invés de referência, encapsulando validações e regras específicas de cada conceito de domínio. Herdam de BaseValueObject garantindo igualdade por valor e imutabilidade, incluindo tipos geoespaciais, documentos brasileiros, endereços, estados de workflow, enumerações tipadas e segurança.

## Categorias

- **[BASE/](./BASE/README.md)** - Classes base para todos value objects
- **[DOCUMENTS/](./DOCUMENTS/README.md)** - CPF CNPJ email telefone e registro CREA
- **[GEO/](./GEO/README.md)** - Tipos geoespaciais GeoPolygon e GeoPoint
- **[ADDRESS/](./ADDRESS/README.md)** - Endereços brasileiros completos
- **[STATUS/](./STATUS/README.md)** - Estados de workflow das entidades
- **[ENUMS/](./ENUMS/README.md)** - Enumerações tipadas do domínio
- **[SECURITY/](./SECURITY/README.md)** - Chaves API e tokens
