# Design Principles - WEBDOCS

## Princípios

**(1) SEO-First** - HTML semântico, meta tags automáticos, sitemap.xml gerado, canonical URLs, structured data (JSON-LD), **(2) Performance** - critical CSS inline, lazy loading images, prefetch links, zero JavaScript by default, **(3) Accessibility** - heading hierarchy correto, alt texts em imagens, color contrast WCAG AA, keyboard navigation, **(4) Mobile-First** - responsive design com Tailwind breakpoints, touch-friendly UI, **(5) Content-Focused** - typography otimizada para leitura, table of contents, breadcrumbs, related pages suggestions, **(6) Developer Experience** - HMR em dev mode, type-safe frontmatter, component library documentada no Storybook.
