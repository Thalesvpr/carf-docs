# Terminology - WEBDOCS

## Glossário

**SSG (Static Site Generation)** - compilação de conteúdo em HTML no build time. **Island** - componente interativo hidratado no cliente. **Slot** - placeholder para children em componentes Astro (similar a Vue slots). **Frontmatter** - metadados YAML no topo de Markdown files. **Content Collection** - grupo de arquivos Markdown com schema Zod. **Layout** - template wrapper para páginas. **Partial Hydration** - hidratar apenas componentes necessários ao invés da página inteira. **MDX** - Markdown com JSX support. **Astro Component** - arquivo `.astro` com HTML-like syntax. **Client Directive** - `client:load`, `client:visible`, `client:idle` controlando hydration.
