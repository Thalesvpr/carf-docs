# RF-029: Vincular Usuário a Equipe

Usuários com role ADMIN podem adicionar usuários existentes a equipes onde seleção de usuários ocorre através de interface de busca e seleção múltipla filtrando por nome email ou role facilitando localização em bases grandes de usuários, usuário pode estar vinculado a múltiplas equipes simultaneamente permitindo participação em projetos cross-funcionais ou comunidades diferentes onde cada vinculação independente e remoção de uma não afeta outras, atualização de permissões ocorre automaticamente após vinculação onde usuário passa a visualizar comunidades atribuídas à nova equipe e filtros de dados são expandidos para incluir escopo da equipe adicionada sem necessidade de logout/login para efetivar mudanças, implementação em módulos GEOWEB e GEOAPI com interface intuitiva permitindo gestão de membros tanto via edição de equipe (adicionar múltiplos usuários) quanto via edição de usuário (adicionar a múltiplas equipes) garantindo flexibilidade de workflow administrativo.
