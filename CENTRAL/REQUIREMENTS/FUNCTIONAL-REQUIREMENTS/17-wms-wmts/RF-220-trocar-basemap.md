# RF-220: Trocar Basemap

O sistema disponibiliza seletor intuitivo de basemap que permite usuário alternar entre diferentes mapas base disponíveis incluindo opções como OpenStreetMap para mapa vetorial detalhado com nomenclatura de ruas e POIs, imagens de satélite de alta resolução para análise de uso real do solo e identificação de edificações, mapas topográficos com curvas de nível para compreensão de relevo, ou mapas temáticos customizados específicos do projeto. A troca entre basemaps ocorre instantaneamente através de simples clique ou tap no seletor, sem necessidade de recarregar página ou aguardar processamento, proporcionando experiência fluida que facilita comparação visual entre diferentes contextos cartográficos durante análise de unidades territoriais. O sistema persiste preferência de basemap selecionada pelo usuário através de cookie ou local storage do navegador, garantindo que em acessos futuros o mapa seja carregado automaticamente com último basemap utilizado, eliminando necessidade de reconfiguração manual repetitiva e respeitando preferências individuais de visualização. Esta funcionalidade reconhece que diferentes tarefas beneficiam-se de diferentes contextos cartográficos, onde análise de limites físicos requer imagens de satélite, planejamento de acesso demanda mapas de ruas detalhados, e análise topográfica necessita representação de relevo, permitindo que cada usuário adapte visualização conforme necessidade específica do momento.
