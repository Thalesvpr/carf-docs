# DASHBOARDS

Dashboards Grafana JSON exports. geoapi-dashboard.json (requests/sec, p50/p95/p99 latency, error rate, endpoints breakdown, database queries, cache Redis). postgres-dashboard.json (connections, transactions/sec, query performance, slow queries table, locks, table sizes, cache hit ratio). infrastructure-dashboard.json (nodes CPU/memory/disk, pods restarts, network I/O, logs errors). Todos com variables environment/timeRange, auto-refresh 30s.
