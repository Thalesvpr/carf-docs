# LGPD Compliance Policy

Política de compliance LGPD (Lei Geral de Proteção de Dados) do CARF garantindo tratamento legal dados pessoais titulares. Consentimento explícito coletado antes processar dados pessoais via checkbox "Concordo com tratamento dados conforme Política Privacidade" não pré-marcado, finalidade específica descrita claramente (dados utilizados exclusivamente para processo regularização fundiária Lei 13.465/2017), e revogação consentimento permitida qualquer momento via botão "Revogar Consentimento" interface usuário. Direitos titulares implementados (confirmação existência tratamento via GET /api/lgpd/my-data retornando JSON dados armazenados, acesso dados pessoais download PDF relatório completo, correção dados desatualizados via formulário edição, anonimização delete lógico marcando deleted_at preservando histórico mas removendo PII, portabilidade export JSON/CSV dados portáteis outro sistema, oposição ao tratamento bloqueando processamento exceto obrigações legais, e revisão decisões automatizadas contestando reprovações automáticas validação CPF). DPO (Data Protection Officer) designado responsável monitorar compliance, receber requisições titulares via dpo@carf.gov.br respondendo em 15 dias úteis, e reportar ANPD incidentes vazamento dentro 72h. Registro de operações tratamento documenta finalidade, base legal, categorias dados, compartilhamento terceiros, e medidas segurança aplicadas, auditado anualmente compliance, e publicado Política Privacidade transparente acessível. Retenção dados limitada período necessário (5 anos pós-conclusão processo legitimação para fins auditoria fiscal), anonimização automática após expiração, e delete seguro via overwrite múltiplos passes impossibilitando recuperação.
