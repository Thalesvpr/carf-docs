# INTEGRATION - Integrações Externas

Documentação de integrações com serviços externos essenciais para funcionamento do sistema CARF incluindo provedor de autenticação centralizada (OAuth2/OIDC) para SSO multi-tenant e banco de dados geoespacial (PostgreSQL + PostGIS) para persistência e queries espaciais.

## Estrutura

- **[KEYCLOAK/](./KEYCLOAK/README.md)** - Provedor centralizado de autenticação OAuth2/OIDC com SSO multi-tenant
- **[DATABASE/](./DATABASE/README.md)** - PostgreSQL 16 + PostGIS 3.4 com Row-Level Security para multi-tenancy
